This project automates the Cleartrip bus booking flow using Playwright.
It simulates a real user journey by selecting a bus from Delhi to Jaipur, choosing a specific travel date, and selecting a particular bus.

The goal of this project is to validate the end-to-end functionality of the Cleartrip bus booking feature.
